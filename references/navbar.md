## 系统状态栏 (Status Bar)

系统级组件，不可自定义内容，仅控制图标颜色。

| 维度 | 选项 |
|------|------|
| 设备 | 手机（主适配）/ 折叠屏 / 平板 |
| 图标模式 | 白色（暗色页面）/ 黑色（亮色页面） |

- 背景透明，与标题栏融合
- 图标颜色通过 CSS `color` + `currentColor` 控制
- SVG 一套，明暗靠 CSS 切换

## 标题栏 (Navigation Bar)

### 类型选择

| 类型 | 适用场景 | 返回按钮 | 标题位置 |
|------|---------|---------|---------|
| 大标题 | 一级页面首屏 | 有 | 左对齐，独占行（上滑自动收起） |
| 中标题 | 一/二级页面 | 有或无（可选） | 左对齐，与图标同行 |
| 小标题 | 通用二级页 | 有 | 居中 |
| 小标题可点击 | 筛选/切换标题 | 无 | 居中胶囊 |

### 调用默认值

做页面时自动从 `navbar-code.md` 取代码。**默认配置：小标题 / 无背景 / 常规上滑**。

### 维度配置

| 维度 | 选项 | 说明 |
|------|------|------|
| 背景 | 无背景（透明）/ 有背景 | 无背景时内容不穿透标题栏 |
| 背景类型 | 常规（不透明实色）/ 高级材质（半透明磨砂玻璃） | 仅「有背景」时选择 |
| 上滑状态 | 常规 / 浮起（玻璃材质图标） | 浮起态标题栏透明，图标用 `.glass-btn` |

### 使用规则

1. **大标题自动收起** — 上滑时自动收起为小标题，是大标题组件的内置行为，不可关闭
2. **状态栏背景跟随标题栏** — 有背景的标题栏，状态栏背景必须与标题栏一致
3. **无背景时内容不穿透** — 内容在标题栏下方开始，不会滚到标题栏区域
4. **浮起态无背景色** — 标题栏透明，图标为玻璃材质，背景为渐变遮罩（顶部不透明→底部透明）

### 大标题结构

```
┌──────────────────────────────────────────┐
│ 功能栏                                    │
│ [返回]                    [图标] [图标] [图标]│
│ 标题区                                    │
│  大标题文字                                │
│  辅助标题（可选）                           │
└──────────────────────────────────────────┘
```

- 上滑时收起为小标题（标题文字移入居中位置，功能栏图标不变）
- 可附带辅助标题（副标题）

### 中标题结构

```
┌──────────────────────────────────────────┐
│ [返回] 标题文字            [图标] [图标] [图标]│
│        辅助标题（可选）                     │
└──────────────────────────────────────────┘
```

- 返回按钮可选（一级页无返回，二级页有返回）
- 无返回时标题从左侧边距开始

### 小标题结构

```
┌──────────────────────────────────────────┐
│ [返回]        标题名称          [图标] [图标]│
│              副标题(可选)                   │
└──────────────────────────────────────────┘
```

- 标题绝对居中，不受左右图标数量影响

### 小标题可点击结构

```
┌──────────────────────────────────────────┐
│ [图标][图标][图标]  ╭可点击文案╮  [图标][图标][图标]│
└──────────────────────────────────────────┘
```

- 中间为可点击胶囊，无返回按钮
- 两侧各最多 3 个图标

### 图标状态

| 状态 | 行为 |
|------|------|
| 默认 | 图标原色 |
| 按压 | opacity 降低 |
| 禁用 | 大幅降低 opacity，不可交互 |

> 精确值见 navbar-code.md `.navbar-icon` 部分。

### 标题栏图标（共 19 种）

返回、取消/关闭、搜索、更多、更多(提醒)、分享、编辑、删除、设置、说明、我的、收藏、订单、扫一扫、确认、侧边栏、上一步、下一步、全选

SVG 代码见 [icons.md](icons.md)。

## 底 Tab 栏 (Bottom Tab Bar)

页面底部主导航组件。

### 何时显示

| 场景 | Tab 栏 |
|------|--------|
| 一级页面（首页等） | 显示 |
| 二级页面（详情等） | 隐藏 |

### 状态

| 状态 | 行为 |
|------|------|
| 未选中 | 灰色图标+文字 |
| 选中 | 橙色图标+文字 |
| 按压 | 图标缩小动画 |

### 角标

任意 Tab 可附加红色圆形角标，用于未读消息/更新提醒。

### Getapps 的 5 个 Tab

| 序号 | 文字 | 说明 |
|------|------|------|
| 0 | Home | 首页 |
| 1 | Games | 游戏 |
| 2 | GameHub | 社区入口 |
| 3 | Top charts | 排行榜 |
| 4 | Profile | 个人中心 |

### 调用方式

```js
const tabBar = new TabBar('#container', {
  tabs: [
    { icon: TAB_ICONS.tab_home, label: 'Home' },
    { icon: TAB_ICONS.tab_games, label: 'Games' },
    { icon: TAB_ICONS.tab_gamehub, label: 'GameHub' },
    { icon: TAB_ICONS.tab_topcharts, label: 'Top charts' },
    { icon: TAB_ICONS.tab_profile, label: 'Profile', badge: 8 }
  ],
  active: 0
});

tabBar.setActive(2);
tabBar.setBadge(4, 3);
tabBar.onSelect = (index) => { ... };
```

## 系统小白条 (Home Bar)

屏幕最底部的系统手势指示条，不可自定义。

| 场景 | 小白条背景 |
|------|-----------|
| 有底 Tab 栏 | 与 Tab 栏背景一致 |
| 无底 Tab 栏 | 透明 |

横条颜色根据页面明暗自动切换。

## 页面骨架模板

标准 Getapps 页面从上到下：

```
┌─────────────────────────────┐
│   系统状态栏                  │  ← 透明，系统占位
├─────────────────────────────┤
│   标题栏                     │  ← 大/中/小标题
├─────────────────────────────┤
│                             │
│   内容区（可滚动）            │
│                             │
├─────────────────────────────┤
│   底 Tab 栏                  │  ← 一级页面显示，二级隐藏
├─────────────────────────────┤
│   系统小白条                  │
└─────────────────────────────┘
```

- 大标题上滑收起为小标题，图标从平面态切为玻璃材质
- 状态栏和标题栏背景与页面一致，无分隔线

> 所有精确样式值（尺寸、间距、字号、颜色、材质参数）见 [navbar-code.md](navbar-code.md)

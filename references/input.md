# Input 输入框

代码：[input-code.md](input-code.md)

---

## 变体维度

| 维度 | 选项 | 说明 |
|------|------|------|
| 类型 | 后缀输入框 / 前缀输入框 | 后缀：输入左fill+图标右；前缀：标签左+输入右对齐 |
| 套卡/通栏 | 套卡 / 通栏 | 套卡有左右内边距；通栏无内边距贴边 |
| 位置 | 独立 / 上 / 中 / 下 | 控制圆角位置，多输入框组合时使用 |
| 背景环境 | 白反灰 / 灰反白 | 根据容器背景色选择（见下方调用规则） |
| 状态 | 默认 / 激活 / 输入 / 报错 | 激活/输入/报错有描边 |

## 结构

### 后缀输入框

```
┌─────────────────────────────────────┐  外容器（套卡有 padding / 通栏无）
│ ┌─────────────────────────────────┐ │  输入框背景（圆角按位置变化）
│ │  输入文字/占位符          [图标] │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 前缀输入框

```
┌─────────────────────────────────────┐
│ ┌─────────────────────────────────┐ │
│ │ 前缀标签  │     输入文字（右对齐）│ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 多行组合

```
┌────────────────────┐  ← 上（顶部圆角）
│ 标签        输入值  │
├────────────────────┤  ← 分隔线
│ 标签        输入值  │  ← 中（无圆角）
├────────────────────┤
│ 标签        输入值  │  ← 下（底部圆角）
└────────────────────┘
```

## 背景环境调用规则

根据输入框所在容器的背景色选择：

| 容器背景 | 选择 | 效果 |
|---------|------|------|
| 白色/浅色（如 `surface`） | 白反灰 | 输入框用灰色背景，形成对比 |
| 灰色/深色/弹窗底（如 `surface_pop_window`） | 灰反白 | 输入框用白色背景，形成对比 |

**原则：输入框背景必须与容器背景产生对比，不能同色融为一体。**

代码调用：白反灰为默认；灰反白加 `grayBg: true` 参数。

## 状态

| 状态 | 触发条件 | 行为 |
|------|---------|------|
| 默认 | 未聚焦 | 无描边 |
| 激活 | 输入框获得焦点 | 整圈描边，圆角恢复为独立形态 |
| 输入 | 正在输入文字 | 同激活态 + 显示文字 |
| 报错 | 调用 setError() | 红色描边 + 底部报错提示文字 |

---

## 参数

| 参数 | 是否必填 | 说明 |
|------|---------|------|
| type | 否 | `'suffix'`（默认）或 `'prefix'` |
| placeholder | 否 | 占位提示文字 |
| label | prefix 时必填 | 前缀标签文字 |
| position | 否 | `'standalone'`（默认）/ `'top'` / `'middle'` / `'bottom'` |
| card | 否 | `true`（默认套卡）/ `false`（通栏） |
| grayBg | 否 | `true` 灰反白 / `false`（默认白反灰） |
| showSuffix | 否 | 是否显示后缀图标 |
| error | 否 | 报错文字，非空时进入报错态 |

---

## 调用方式

```js
// 后缀输入框 — 套卡/独立/白反灰（最常用）
new InputField('#container', {
  placeholder: '请输入内容',
  showSuffix: true
});

// 前缀输入框
new InputField('#container', {
  type: 'prefix',
  label: '手机号',
  placeholder: '请输入'
});

// 通栏/灰反白（弹窗内使用）
new InputField('#container', {
  placeholder: '请输入内容',
  card: false,
  grayBg: true
});

// 多行组合
new InputField('#container', { type: 'prefix', label: '姓名', position: 'top' });
new InputField('#container', { type: 'prefix', label: '电话', position: 'middle' });
new InputField('#container', { type: 'prefix', label: '地址', position: 'bottom' });
```

> 所有精确样式值（颜色、间距、字号、圆角）见 [input-code.md](input-code.md)

# Dialog 弹窗

弹窗内容每次不同，不提供固定代码文件。做页面时按以下规格拼装，按钮从 [button-code.md](button-code.md) 取代码。

---

## 通用规格

| 属性 | 值 |
|------|------|
| 容器宽度 | max-width: 368px, width: 100%（屏幕宽度决定弹窗宽度，内容适应弹窗） |
| 圆角 | 36 `{radius.big}` |
| 容器内边距 | 24/0/24/0（上下 24，左右由子区域各自控制） |
| 区域间距 | 16 |
| 背景 | `surface_pop_window`（Light: #f7f7f7 / Dark: #242424） |
| 遮罩 | `mask`（Light: rgba(0,0,0,0.2) / Dark: rgba(0,0,0,0.6)） |
| 位置 | 页面垂直水平居中 |
| 材质 | Default（纯色）/ Frosted（毛玻璃）/ Glass（禁用） |

## 弹窗结构

```
┌────────────────────────────────┐  ← 368px, r=36
│  TitleArea (24px)               │  标题区
│  ─────────────────────────────  │
│  Body                           │  正文区 pad: 0/28
│  ─────────────────────────────  │
│  备注文字（可选）                 │  pad: 0/28
│  ─────────────────────────────  │
│  CheckboxArea（可选）            │  pad: 0/28
│  ─────────────────────────────  │
│  ButtonArea                     │  按钮区 pad: 8/24/0/24
└────────────────────────────────┘
```

## 各区域规格

### TitleArea 标题区

| 属性 | 值 |
|------|------|
| 高度 | 24 |
| 标题字号 | `font.title4` fs=18 fw=380 |
| 标题色 | `on_surface` |
| 对齐 | 居中 |

### Body 正文区

| 属性 | 值 |
|------|------|
| 左右内边距 | 28 |
| 正文字号 | `font.body1` fs=16 fw=330 |
| 正文色 | `on_surface_secondary` |
| 对齐 | 单行居中，多行居左 |
| 行间距 | 默认 |

### 备注文字（可选）

| 属性 | 值 |
|------|------|
| 左右内边距 | 28 |
| 字号 | `font.footnote1` fs=13 fw=330 |
| 颜色 | `on_surface_quaternary` |
| 对齐 | 单行居中，正文多行时居左 |

### CheckboxArea 勾选区（可选）

| 属性 | 值 |
|------|------|
| 左右内边距 | 28 |
| Checkbox 高度 | 22 |
| Checkbox 文字 | fs=16 fw=330 `on_surface_secondary` |

### ButtonArea 按钮区

| 属性 | 值 |
|------|------|
| 内边距 | 8/24/0/24 |
| 按钮 | 使用 BaseButton Large，从 [button-code.md](button-code.md) 取代码 |
| 按钮间距 | 12（上下排列时） |
| 按钮布局 | 单按钮满宽 / 双按钮上下排列（Primary 在上，Secondary 在下）/ 2 个以上按钮竖向堆叠 |

## 弹窗类型

### ConfirmDialog 确认弹窗

最常用的弹窗，标题+正文+按钮。

| 变体 | 说明 |
|------|------|
| 单行正文 | 正文单行居中，备注居中 |
| 多行正文 | 正文多行居左，备注居左 |
| 带勾选 | 正文下方有 Checkbox（如"不再提醒"） |
| 2个以上按钮 | 按钮竖向堆叠 |
| 通知弹窗 | 说明通知型，用户无需更多操作 |

### AlertDialog 警示弹窗

带遮罩和材质效果的确认弹窗。

| 变体 | 说明 |
|------|------|
| Light Default | 白色纯色底 |
| Dark Default | 深色纯色底 |
| Light Frosted | 白色毛玻璃 |
| Dark Frosted | 深色毛玻璃 |

### WarningDialog 警告弹窗

用于卸载、删除等危险操作确认。与 ConfirmDialog 的区别：标题区上方有应用图标。

```
┌────────────────────────────────┐
│        [App图标 56×56]          │
│           标题                  │
│  ─────────────────────────────  │
│  正文                           │
│  ─────────────────────────────  │
│  备注文字                       │
│  ─────────────────────────────  │
│  [取消]  [卸载(Destructive)]    │  ← 左右并排 50h, gap=12
└────────────────────────────────┘
```

- 图标：56×56，居中，在标题上方
- 按钮布局：**左右并排**（不是上下堆叠），每个 154px 宽，50px 高，间距 12
- Destructive 按钮用 `error` 色文字

### GuideDialog 引导弹窗

带插图的引导弹窗。与 ConfirmDialog 的区别：标题区上方有插图区域。

```
┌────────────────────────────────┐
│     [插图区域 320×88]           │  ← 宽度随弹窗，高度按内容
│           标题                  │
│  ─────────────────────────────  │
│  正文                           │
│  ─────────────────────────────  │
│  备注文字                       │
│  ─────────────────────────────  │
│  [取消]  [主操作]               │  ← 左右并排
└────────────────────────────────┘
```

- 插图区域在最顶部（标题上方），宽度 calc(100% - 48px)，高度按比例（aspect-ratio: 320/88）
- 按钮布局：**左右并排**，每个 154px 宽

### InputDialog 输入弹窗

带输入框的弹窗。结构：标题 + 输入框 + 按钮，无正文区。

```
┌────────────────────────────────┐
│           标题                  │  pad: 0/36
│  ─────────────────────────────  │
│  [输入框 320×56]                │  pad: 0/24, r=12
│  ─────────────────────────────  │
│  [取消]  [确定]                 │  ← 左右并排
└────────────────────────────────┘
```

- 标题内边距：0/36（比通用的 0/28 更大）
- 输入框：宽 320，高 56，内边距 0/24
- 按钮布局：**左右并排**

### ProgressDialog 进度弹窗

加载/下载进度弹窗。结构紧凑，无标题无按钮。

```
┌────────────────────────────────┐
│  [Loading图标] 正在更新请稍候...  │  ← 图标 24×24 + 文字 fs=17 fw=380, 水平排列, pad: 0/24
└────────────────────────────────┘
```

- 高度：72
- 布局：水平排列，图标 24×24 + 文字，间距 8
- 文字：`font.headline1` fs=17 fw=380，颜色 `on_surface_secondary`
- 无标题、无按钮

### SelectionDialog 选择弹窗

带选项列表的弹窗。

```
┌────────────────────────────────┐
│           标题                  │  pad: 0/36
│  ─────────────────────────────  │
│  ○ 选项一                       │  每行 56h, pad: 14/28
│  ○ 选项二                       │
│  ● 选项三（选中）          ✓    │  ← 选中项右侧有勾
│  ─────────────────────────────  │
│  [取消]  [确定]                 │  ← 左右并排
└────────────────────────────────┘
```

- 标题内边距：0/36
- 每个选项行：高度 56，内边距 14/28，文字 fs=17 fw=380 `on_surface`
- 选中项文字色：`primary`（蓝色），右侧勾选图标 20×20
- 按钮布局：**左右并排**

### PermissionDialog 权限弹窗

权限申请弹窗。与其他弹窗的关键区别：**图标在标题上方**。

```
┌────────────────────────────────┐
│        [权限图标 32×32]          │  ← 居中，在标题上方
│           标题                  │
│  ─────────────────────────────  │
│  [定位选项区域]                  │  ← 可选（如模糊定位/精准定位）
│  ─────────────────────────────  │
│  备注文字                       │
│  ─────────────────────────────  │
│  [拒绝]                         │  ← 多个按钮竖向堆叠
│  [本次使用允许]                  │
│  [仅在使用中允许]                │
│  [始终允许]                     │
└────────────────────────────────┘
```

- 权限图标：32×32，r=8，**居中，在标题上方**
- 可选的选项区域（如定位弹窗有模糊/精准选择）：pad 0/20，两列布局
- 按钮：多个按钮竖向堆叠
- 权限图标共 30+ 种（32×32，r=8）

---

## 按钮布局汇总

| 弹窗类型 | 按钮布局 | 说明 |
|---------|---------|------|
| ConfirmDialog 单/多行 | 上下堆叠 | Primary 在上，Secondary 在下 |
| ConfirmDialog 2个以上 | 上下堆叠 | 全部竖排 |
| ConfirmDialog 通知 | 单按钮满宽 | 仅一个"我知道了" |
| WarningDialog | **左右并排** | 取消在左，Destructive 在右 |
| GuideDialog | **左右并排** | 取消在左，Primary 在右 |
| InputDialog | **左右并排** | 取消在左，确定在右 |
| SelectionDialog | **左右并排** | 取消在左，确定在右 |
| PermissionDialog | 上下堆叠 | 多个权限选项 |
| ProgressDialog | 无按钮 | — |

> 左右并排时每个按钮 154px 宽，50px 高，间距 12。

---

## 布局位置

弹窗在页面中垂直水平居中显示，遮罩覆盖全屏。

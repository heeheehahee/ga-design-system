代码：[dialog-code.md](dialog-code.md)

---

## 弹窗类型选择

| 场景 | 用哪个类型 |
|------|-----------|
| 通用确认/提示 | ConfirmDialog |
| 危险操作（卸载/删除） | WarningDialog |
| 功能引导（带插图） | GuideDialog |
| 需要用户输入文字 | InputDialog |
| 加载/等待中 | ProgressDialog |
| 让用户选一项 | SelectionDialog |
| 系统权限申请 | PermissionDialog |
| 带材质效果的确认 | AlertDialog |

---

## 通用结构

```
┌────────────────────────────────┐
│  TitleArea                      │  标题区
│  ─────────────────────────────  │
│  Body                           │  正文区
│  ─────────────────────────────  │
│  备注文字（可选）                 │
│  ─────────────────────────────  │
│  CheckboxArea（可选）            │
│  ─────────────────────────────  │
│  ButtonArea                     │  按钮区
└────────────────────────────────┘
```

---

## 位置

| 位置 | 适用场景 |
|------|---------|
| 居中（默认） | 大多数弹窗 |
| 底部吸底 | `position: 'bottom'`，贴近屏幕底部 |

---

## 内容滚动规则

弹窗高度超过屏幕高度 70% 时，Body 正文区出现垂直滚动条（使用 `.custom-scrollbar`），标题区和按钮区固定不滚动。

---

## 内嵌组件规则

弹窗内放置输入框等子组件时，子组件的外边距以弹窗容器边距为准，不使用子组件自身的默认边距。输入框使用通栏模式（`input-wrap--full`）。

## 弹窗类型详情

### ConfirmDialog 确认弹窗

最常用。标题 + 正文 + 按钮。

| 变体 | 差异 |
|------|------|
| 单行正文 | 正文/备注居中 |
| 多行正文 | 正文/备注居左 |
| 带勾选 | 正文下方有 Checkbox（如"不再提醒"） |
| 多按钮 | 2 个以上按钮竖向堆叠 |
| 通知型 | 仅一个"我知道了"按钮 |

### WarningDialog 警告弹窗

危险操作确认。与 Confirm 区别：**标题上方有应用图标**。

```
┌────────────────────────────────┐
│        [App图标]                │
│         标题                    │
│  正文                           │
│  [取消]  [卸载(Destructive)]    │  ← 左右并排
└────────────────────────────────┘
```

- Destructive 按钮用 `error` 色文字

### GuideDialog 引导弹窗

带插图。与 Confirm 区别：**标题上方有插图区域**。

```
┌────────────────────────────────┐
│     [插图区域]                   │
│         标题                    │
│  正文                           │
│  [取消]  [主操作]               │  ← 左右并排
└────────────────────────────────┘
```

### InputDialog 输入弹窗

带输入框。结构：标题 + 输入框 + 按钮，无正文区。

```
┌────────────────────────────────┐
│         标题                    │
│  [输入框]                       │
│  [取消]  [确定]                 │  ← 左右并排
└────────────────────────────────┘
```

### ProgressDialog 进度弹窗

加载等待态。胶囊形状，无标题无按钮。

```
┌────────────────────────────────┐
│  [Loading图标] 正在更新请稍候...  │  水平排列
└────────────────────────────────┘
```

### SelectionDialog 选择弹窗

带选项列表。

```
┌────────────────────────────────┐
│         标题                    │
│  ○ 选项一                       │
│  ○ 选项二                       │
│  ● 选项三（选中）          ✓    │  ← 选中项右侧有勾
│  [取消]  [确定]                 │  ← 左右并排
└────────────────────────────────┘
```

- 选中项文字色切换为 `primary`

### PermissionDialog 权限弹窗

权限申请。关键区别：**图标在标题上方**，按钮为多个权限选项竖向堆叠。

```
┌────────────────────────────────┐
│        [权限图标]                │
│         标题                    │
│  [定位选项区域]（可选）           │
│  备注文字                       │
│  [拒绝]                         │
│  [本次使用允许]                  │
│  [仅在使用中允许]                │
│  [始终允许]                     │
└────────────────────────────────┘
```

### AlertDialog 警示弹窗

带材质效果的确认弹窗。

| 变体 | 说明 |
|------|------|
| Light Default | 白色纯色底 |
| Dark Default | 深色纯色底 |
| Light Frosted | 白色毛玻璃 |
| Dark Frosted | 深色毛玻璃 |

---

## 按钮布局规则

| 弹窗类型 | 按钮布局 | 说明 |
|---------|---------|------|
| ConfirmDialog | 上下堆叠 | Primary 在上，Secondary 在下 |
| ConfirmDialog 通知 | 单按钮满宽 | 仅"我知道了" |
| WarningDialog | **左右并排** | 取消在左，Destructive 在右 |
| GuideDialog | **左右并排** | 取消在左，Primary 在右 |
| InputDialog | **左右并排** | 取消在左，确定在右 |
| SelectionDialog | **左右并排** | 取消在左，确定在右 |
| PermissionDialog | 上下堆叠 | 多个权限选项 |
| ProgressDialog | 无按钮 | — |

## 调用方式

```js
// 确认弹窗
new Dialog({ title: '标题', body: '正文', buttons: [...] })

// 警告弹窗 — 横排按钮
new Dialog({ title: '卸载"应用"', body: '...', buttonsLayout: 'horizontal', buttons: [...] })

// 选择弹窗
new Dialog({ title: '选择', options: [{text:'选项一'}, ...], buttonsLayout: 'horizontal', buttons: [...] })

// 进度弹窗
new Dialog({ type: 'progress', body: '正在更新请稍候...' })

// 底部吸底
new Dialog({ position: 'bottom', ... })
```

> 所有精确样式值（尺寸、间距、字号、颜色）见 [dialog-code.md](dialog-code.md)

## 系统状态栏 (Status Bar)

系统级组件，不可自定义内容，仅控制图标颜色。

### 设备规格

| 设备 | 宽度 | 高度 | 说明 |
|------|------|------|------|
| 手机 | 392 | 46 | Getapps 主要适配设备 |
| 折叠屏 | 741 | 38 | 含前摄挖孔区域 |
| 平板 | 1219 | 34 | — |

### 变体

| 变量 | 选项 | 说明 |
|------|------|------|
| 类型 | 电池在内 / 电池在外 | 电池在外时显示电量百分比 |
| 模式 | 白色图标 / 黑色图标 | 白色用于暗色页面，黑色用于亮色页面 |

### 实现方式

- 背景：**透明**，与标题栏/页面背景融合
- 图标颜色：使用 `currentColor`，通过 CSS `color` 属性控制
- 亮色页面：`color: var(--text-strong)` → 黑色图标
- 暗色页面：`color: var(--text-strong)` → 白色图标
- SVG 路径一套，明暗靠 CSS 切换，无需两张图

> 实现时状态栏为系统占位区域，高度固定 46px，页面内容从状态栏下方开始。

## 标题栏 (Navigation Bar)

来源：Figma ga_component node 224:4859（通用）、224:4976（高级材质）

---

### 变体维度

| 维度 | 选项 | 说明 |
|------|------|------|
| 标题类型 | 大标题 / 中标题 / 小标题 / 小标题可点击 | 4 种标题排版 |
| 返回按钮 | 有 / 无 | **仅中标题**，其他类型固定 |
| 背景 | 有背景 / 无背景（透明） | 决定内容是否穿透标题栏 |
| 背景类型 | 常规 / 高级材质 | **有背景时**选择 |
| 上滑状态 | 常规 / 浮起 | 浮起态无背景，图标为玻璃材质 |

### 标题类型总览

| 类型 | 高度 | 字号/字重 | 返回按钮 | 标题位置 | 适用场景 |
|------|------|----------|---------|---------|----------|
| 大标题 | 116 | 32/330 | 有 | 左对齐，独占行 | 一级页面首屏 |
| 中标题 | 56 | 26/380 | 有/无（可选） | 左对齐，与图标同行 | 一/二级页面 |
| 小标题 | 56 | 18/380 | 有 | 居中 | 通用 |
| 小标题可点击 | 56 | 14/380 | 无 | 居中胶囊 | 可点击筛选标题 |

### 调用默认值

做页面需要标题栏时，自动从 `navbar-code.md` 取代码。**默认配置：小标题 / 无背景 / 常规上滑**。调用完成后提示用户可选其他样式：大标题（自动收起）、中标题（有/无返回）、小标题可点击（胶囊）、有背景（常规/高级材质）、浮起态。

### 使用规则

1. **大标题自动收起**：大标题上滑时自动收起为小标题（116→56），这是大标题组件的**内置行为**，不是可选功能
2. **状态栏背景跟随标题栏**：有背景的标题栏，状态栏背景必须与标题栏一致，视觉融为一体
3. **无背景时内容不穿透**：选择无背景（透明）时，内容在标题栏下方开始，不会滚到标题栏区域
4. **浮起态无背景色**：浮起态时标题栏本身透明，图标按钮为玻璃材质（`.glass-btn`），背景为渐变（顶部不透明→底部透明）
5. **颜色使用 Token**：标题文字和图标颜色使用 `var(--on_surface)`，辅助标题使用 `var(--on_surface_tertiary)`

### 背景类型参数

#### 常规背景

不透明实色，使用 `var(--surface)` Token。

| 模式 | 值 |
|------|------|
| 亮色 | `#ffffff`（不透明） |
| 暗色 | `#000000`（不透明） |

#### 高级材质背景

半透明磨砂玻璃，内容可透出。来源：Figma 样式 `Frosted 混色`。

| 参数 | 亮色模式 | 暗色模式 |
|------|---------|---------|
| 填充层 1 | `rgba(249,249,249,0.2)` | `rgba(0,0,0,0.46)` |
| 填充层 2 | `rgba(247,247,247,0.7)` | `rgba(0,0,0,0.32)` |
| 模糊 | `backdrop-filter: blur(70px)` | 同左 |

#### 浮起态背景

标题栏本身透明，图标按钮为 `.glass-btn` 玻璃材质。背景层为渐变遮罩：

| 参数 | 亮色模式 | 暗色模式 |
|------|---------|---------|
| 渐变 | `rgba(255,255,255,1)` → `transparent` | `rgba(0,0,0,1)` → `transparent` |
| 方向 | 从上到下 | 同左 |

### 通用属性

| 属性 | 值 | 说明 |
|------|------|------|
| 宽度 | 392（填充页面） | 设计基准宽度 |
| 图标板尺寸 | 44×44 | 触摸区域 |
| 图标 SVG | 40×40 viewBox | 图形占中心约 28×28 |
| 图标间距 | 8 | 多图标之间 |
| 标题/图标颜色 | `on_surface` | Token，Light `#000000` / Dark `rgba(255,255,255,0.95)` |
| 辅助标题颜色 | `on_surface_tertiary` | Token，Light `rgba(0,0,0,0.6)` / Dark `rgba(255,255,255,0.6)` |

### 图标状态

| 状态 | 亮色模式 | 深色模式 |
|------|---------|---------|
| 默认 | `#000000` 100% | `rgba(255,255,255,0.95)` |
| 按压 | 底色 `rgba(0,0,0,0.3)` | 底色 `rgba(255,255,255,0.3)` |
| 禁用 | `#000000` 30% | `rgba(255,255,255,0.4)` |

### 大标题（116px）

两层结构：上层功能栏 + 下层标题区。**上滑自动收起为小标题。**

```
┌──────────────────────────────────────────┐  ← 392 × 116
│ 功能栏 (56px)  padding: 6/12/6/12        │
│ ┌────┐                    ┌────┬────┬────┐│
│ │返回 │                    │ 图 │ 图 │ 图 ││
│ │44×44│                    │44×44    ×3  ││
│ └────┘                    └────┴────┴────┘│
│ 标题区 (60px)  padding: 8/28/10/28        │
│  大标题  fs=32 fw=330                      │
│  辅助标题 fs=14 fw=380（可选）               │
└──────────────────────────────────────────┘
```

| 属性 | 值 |
|------|------|
| 总高度 | 116 |
| 功能栏 | 56px，HORIZONTAL，SPACE_BETWEEN，gap=48 |
| 功能栏内边距 | 6/12/6/12 |
| 左侧 | 1 个图标板（返回按钮） |
| 右侧 | 最多 3 个图标板，间距 8 |
| 标题区 | 60px，VERTICAL |
| 标题区内边距 | 8/28/10/28 |
| 主标题 | MiSans fs=32 fw=330，色 `on_surface` |
| 辅助标题 | MiSans fs=14 fw=380，色 `on_surface_tertiary` |

### 滚动交互

大标题上滑自动收起为小标题（**内置行为**，调用大标题组件即自动包含）。

| 行为 | 标题栏 | 高度 |
|------|--------|------|
| 页面顶部 | 大标题 | 116 |
| 内容上滑 | 小标题（标题文字居中） | 56 |
| 回到顶部 | 大标题 | 116 |

**切换规则：**
- 大标题文字移入小标题栏居中位置
- 功能栏左右图标保持不变
- 状态栏高度不变（始终 46px）
- 二级页面可直接使用小标题/中标题，不需要大标题收起

## 底 Tab 栏 (Bottom Tab Bar)

页面底部主导航组件，Tab 数量和内容可配置。图标 SVG 见 [icons.md](icons.md)。

### 组件规格

| 属性 | 值 |
|------|------|
| 宽度 | 填充页面（392px 基准） |
| 高度 | 65 |
| 内边距 | 0/8/0/8 |
| Tab 布局 | 等分宽度，水平排列 |
| 图标尺寸 | 28×28 |
| 文字 | fs=11 fw=380 |
| 图标与文字间距 | 2 |
| 背景 | `var(--surface)` |
| 底部保留 | 16px（系统小白条 homebar 高度），背景同 Tab 栏 |

### 状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 未选中 | `on_surface_tertiary`（灰色） | 亮暗模式各自的灰色 |
| 选中 | `#FF6900`（橙色） | 固定橙色，不区分亮暗模式 |
| 按压 | 图标缩小 `scale(0.85)` | `transition: transform 0.15s ease` |

> 所有 Tab 图标使用 `fill="currentColor"`，通过 CSS `color` 控制选中/未选中色。选中色固定为 `#FF6900`。

### 角标

任意 Tab 可附加红色角标，用于未读消息/更新提醒：
- 尺寸：min-width 16，height 16，圆角 999
- 背景：`#FA382E`（error 色）
- 文字：10px，白色，居中
- 位置：图标右上角偏移

### 调用方式

```js
const tabBar = new TabBar('#container', {
  tabs: [
    { icon: TAB_ICONS.tab_home, label: 'Home' },
    { icon: TAB_ICONS.tab_games, label: 'Games' },
    { icon: TAB_ICONS.tab_gamehub, label: 'GameHub' },
    { icon: TAB_ICONS.tab_topcharts, label: 'Top charts' },
    { icon: TAB_ICONS.tab_profile, label: 'Profile', badge: 8 }
  ],
  active: 0  // 默认选中第一个
});

tabBar.setActive(2);          // 切换选中
tabBar.setBadge(4, 3);        // 给第5个Tab设置角标
tabBar.onSelect = (index) => { ... };
```

## 系统小白条 (Home Bar / System Line)

屏幕最底部的系统手势指示条，不可自定义，仅控制颜色模式。

### 规格

| 属性 | 值 |
|------|------|
| 容器宽度 | 392（填充页面） |
| 容器高度 | 16 |
| 横条宽度 | 152 |
| 横条高度 | 4 |
| 横条圆角 | 4（全圆角） |
| 位置 | 容器水平居中 |

### 颜色

| 模式 | 横条颜色 | 说明 |
|------|---------|------|
| 亮色页面 | `rgba(0,0,0,0.8)` | 黑色 80% |
| 暗色页面 | `rgba(255,255,255,0.7)` | 白色 70% |

### 背景规则

| 场景 | 小白条背景色 | 说明 |
|------|------------|------|
| 有底 Tab 栏 | 与底 Tab 栏背景一致 | 视觉上与 Tab 栏融为一体 |
| 无底 Tab 栏 | 透明 | 与页面内容背景融合 |

> 横条颜色根据页面明暗自动切换。小白条始终位于页面最底部。

## 页面骨架模板

标准 Getapps 页面从上到下：

```
┌─────────────────────────────┐
│   系统状态栏 (46px)          │  ← 透明背景，系统占位
├─────────────────────────────┤
│   标题栏 (56~116px)         │  ← 大/中/小标题
├─────────────────────────────┤
│                             │
│   内容区                     │  ← 页面左右边距 20px
│   (可滚动)                   │     卡片内边距 16px
│                             │
├─────────────────────────────┤
│   底 Tab 栏 (65px)          │  ← 一级页面显示，二级隐藏
├─────────────────────────────┤
│   系统小白条 (16px)          │  ← 系统手势指示条
└─────────────────────────────┘
```

- 页面宽度：392px（设计基准）
- 上滑时大标题收起为小标题（116→56），图标板子从平面态切换为玻璃材质
- 状态栏和标题栏背景与页面一致，无分隔线
- 底 Tab 栏固定底部，高度 65px，一级页面显示，二级页面隐藏
- 系统小白条始终在最底部，高度 16px，不可自定义
